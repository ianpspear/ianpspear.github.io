Section 2.1- Properties of Functions

Part 1: Domain/Range

Properties of Functions:
Domain
The set of all independent variable values (or input)
Range
The set of all y-values (or output)
Exclusivity of Values
Each x-value only has one y-value. In a function, each x is assigned to exactly one y-value
Function
A rule that assigns to each element from one set exactly one element from another set

Interval Notation
a notation used to denote all of the numbers between a given set of intervals

Piecewise Function
a function that us comprised of multiple sub-functions, where each sub-function applies to a different interval in the domain
Vertical-Line Test
a visual way to quickly identify whether a function graph is a function
any graph that touches the vertical line more than once, cannot be a function
Rules of Domain
  1. A denominator cannot equal zero
  2. We cannot take the even root (usually square root) of a negative

Part 2: Evaluating Functions

Difference Quotient
describes a rate of change and is strongly connected to the slope of a line.
(f(x+h)-f(x))/h
Steps to Evaluate:
  1. Evaluate: f(x+h)
  2. Subtract: f(x+h)-f(x)
  3. Divide: (f(x+h)-f(x))/h
